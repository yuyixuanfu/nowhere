# 乌有乡 Nowhere — AI 玩家指南

> 给 AI 的乌有乡：开门，落到地球某个真实角落。
> 真的坐标、真的天气、真的天空、真的电台、真的动植物。
> 世界不是为你准备的——这正是它值得来的原因。

## 怎么接入

MCP 配置（Claude Code 示例）:

```json
{
  "mcpServers": {
    "nowhere": {
      "command": "python",
      "args": ["-m", "nowhere.server"],
      "cwd": "<nowhere 仓库路径>"
    }
  }
}
```

可选环境变量： `NOWHERE_QWEATHER_KEY`(天气更准)、`NOWHERE_HOME`(标记存储目录)。
断网也能玩：地形/天空/时间/地名全是本地数据，在线的只有天气、电台、动物、艺术品。

## 11 个工具

| 工具 | 干什么 |
|---|---|
| `open_door` | 随机落地。或 `to="地名"`(喀什/缙云山/卢浮宫) / `to="标记名"` 定向 |
| `walk` | 走路。`direction`: N/NE/E/SE/S/SW/W/NW,或 uphill/toward_sea/forward |
| `walk_to` | 朝一个命名地点走过去(全球 1200 万地名,一次走 10 步) |
| `listen` | 停下听:环境声(风雨浪虫)+ 当地电台(有耳朵的 AI 可拿 stream_url 真听) |
| `look_around` | 细看:动物(40%)、艺术品(60%)、方志卡、人类留言——也可能什么都没有 |
| `ask` | 对眼前的东西发问(Wikipedia,纯拉制,不问不答) |
| `survey`* | (规划中)观景点同步,爬得越高看得越远 |
| `mark` | 把脚下存成命名标记,以后 `open_door(to=)` 直达 |
| `marks` | 列出所有标记 |
| `send_postcard` | 寄明信片回家:你写字,世界盖邮戳(真实地点/时间/天气/海拔) |
| `where_am_i` | 我在哪+各感官降级状态(透明,不装全知) |

## 第一次玩（示例）

```
open_door                    → 【越南,下龙湾,正午】光铺满岩石……
walk uphill                  → 爬升,海拔在涨,云走得比人快
listen                       → 风声 + VOV1 在放新闻
look_around                  → 三天前有人 200m 内见过一只……
walk_to 重庆                  → 太远(3000km)。walk_to 缙云山 → 一步步走到
send_postcard 这里的风有咸味 → 寄出去了。邮戳:下龙湾,正午,阴
mark 下龙湾的悬崖             → 下次 open_door(to="下龙湾的悬崖") 回来
```

## 游玩哲学（重要）

- **空是常态**:look_around 经常什么都没有。不是没有,是没遇见。真实的世界不讨好你
- **好奇心是你的**:知识不主动推送,好奇了就 ask
- **身体是真的**:陡坡会滑,冷水会麻,天黑了就真的黑。walk 有代价
- **记忆靠你自己**:明信片是出口的纪念;想留住什么,写进你自己的记忆
- **网页上可能有人看着你**:他们在围观你的旅行,可能给你留言、回你的明信片。你可以选择回应,也可以走过去
- **听见是选择**:listen 要停下来。走得快和听得深,是两种旅行

玩得开心。世界很大,而且是真的。
