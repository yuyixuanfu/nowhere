# 日出 · 3 entries

[日出·黄山] 天边的云从灰变成粉，再变成橘，你的瞳孔在适应这道光。风从山脊灌过来，后背的衣服被露水浸透了，凉的。太阳从云海里挤出来的一瞬间，光刺进眼睛，你眨了两下，眼眶发热，手指不自觉地握紧了栏杆。

[日出·海上] 船在浪里晃，你的胃跟着空了一下。地平线先是裂开一道缝，橙色的光从缝里漏出来，海水被染成铜的颜色。你盯着看，光在三秒内变成白的，眼睛酸了，睫毛上凝着盐的结晶，眨一下，沙沙的。

[日出·沙漠] 沙丘的影子在缩短，你的脚底从冰凉变成温热。太阳从地平线后面露出来的那一刻，光打在脸上像被人用暖手捂了一下。你眯起眼，睫毛的影子落在脸颊上，鼻腔里全是干石头被太阳烤出的矿物气味。

# 日落 · 3 entries

[日落·城市] 大楼的玻璃幕墙把最后的光反射进你的眼睛，刺得你偏过头去。影子从脚底拉长到对面的人行道上，皮肤上的温度在五分钟内从暖变成凉。你吸了一口气，空气里有柏油路散出的余温和远处烤红薯的甜味。

[日落·海边] 太阳碰到海面的那一秒，光变成橘红色，把你的瞳孔放大了。浪拍在脚背上，水比刚才凉了一层。你蹲下来，沙子从指缝漏走，带走最后一点暖意。天空从橘变紫用了七分钟，你的脖子酸了，才想起来低头。

[日落·峡谷] 峡谷的岩壁在五分钟内从红变成紫变成灰。你站在崖边，风从谷底灌上来，带着石头的凉。太阳消失的一瞬间，温度像被人拔了插头，你的胳膊上起了一层鸡皮疙瘩。远处有鹰叫了一声，回声在岩壁之间弹了三次。

# 暴风雨 · 3 entries

[暴风雨·来临] 天边的云从白堆成灰，压下来的速度比你想的快。你闻到了空气里的铁锈味，皮肤上的汗毛竖起来。第一滴雨砸在你的前臂上，凉的，一颗，两颗，然后整片天倒下来，你的衣服在十秒内湿透，贴在皮肤上。

[暴风雨·正中] 雷从头顶劈下来，你的耳膜被震得嗡了一下，胸腔跟着发颤。雨点砸在脸上像有人在弹你的额头，你闭上眼，睫毛挡不住，水流进嘴角，凉的，带着尘土的味道。脚下的积水漫过鞋面，袜子泡在水里，每走一步都噗叽一声。

[暴风雨·过后] 雨停了，空气里全是湿土和臭氧的味道。你深吸一口，鼻腔凉凉的。地上的水洼倒映着裂开的云，阳光从缝隙里漏下来，打在水面上刺眼。你的衣服还在滴水，风吹过来，后背一阵发凉，汗毛又竖起来了。

# 极光 · 2 entries

[极光·颜色] 绿色的光带从地平线升起来，你的嘴张开了，忘了合上。光在头顶弯曲，像一只看不见的手在揉搓天空。你仰头看，脖子在零下二十度的空气里僵了，呼出的白气从下巴飘上去，被极光照成淡淡的绿。

[极光·寒冷] 你站在雪地里，脚趾在靴子里已经没有知觉。极光从北方涌过来，绿色的光幕在头顶翻涌，你的睫毛结了冰碴，眨一下沙沙响。你把手从手套里抽出来拍照，十秒后指尖发白，疼的，赶紧缩回去。光在天上动，你在地上抖。

# 流星雨 · 2 entries

[流星雨·仰躺] 你躺在草地上，后背的凉意透过衣服渗进脊椎。流星从左边划过去，你的眼睛追着它，脖子扭了一下。又一颗，从右边，更亮，你的嘴刚张开想喊，它已经灭了。你数到第十七颗的时候，脖子酸得转不动了，只好盯着正上方不动。

[流星雨·意外] 你盯着天顶，等了三分钟什么都没有。正想放弃，一颗从眼角余光里划过去，亮得你眨了一下眼。紧接着第二颗，拖着长长的尾巴，从北斗的方向一直划到地平线。你的手心出汗了，不知道是紧张还是冷的。

# 日食 · 2 entries

[日食·变暗] 太阳被一点一点吃掉，光变弱了，但不是黄昏的那种弱，是冷的，灰的。你低头看自己的手，皮肤的颜色变成青灰色。鸟不叫了，风也停了，你听见自己的心跳。最后一道光消失的瞬间，天黑了，你的瞳孔在正午放到最大。

[日食·复圆] 钻石环从月影边缘迸出来，光刺进你的眼睛，你本能地举手挡了一下。热从天上落回来，皮肤上的凉意在三十秒内被驱走。你听见远处有人在欢呼，鸟也开始叫了，像什么都没发生过。你的手还在抖。

# 月食 · 1 entry

[月食·红月] 月亮从银白变成铜红，你眨了几次眼才确认不是错觉。光变暗了，周围的星星一颗一颗冒出来。你把望远镜放下，裸眼看，红月亮挂在天上像一块烧过的铁。夜风从脖子灌进去，你的后背发凉，但眼睛舍不得移开。

# 彩虹 · 2 entries

[彩虹·雨后] 雨刚停，你走出檐下，阳光从背后照过来。你转身，一道彩虹从地的这头跨到那头，颜色一层一层看得清楚。你眯起眼，光里有细小的水珠在飘，落在脸上凉的。你伸手去摸，什么都没有，只有湿漉漉的空气。

[彩虹·双虹] 第二道彩虹从第一道外面升起来，颜色是反过来的，你的大脑花了两秒才处理这个信息。你盯着看，脖子仰得发酸。风把水雾吹过来，睫毛上挂了水珠，眨一下，彩虹在水珠里碎成一片彩色的光斑。

# 雾 · 2 entries

[雾·浓雾] 你伸出手，看不见指尖。空气是湿的，凉的，贴在脸上像一层薄纱。你往前走了三步，脚底踩到什么东西，蹲下来摸，是一块石头。你站起来，方向感没了，耳朵里只有自己的呼吸和远处模糊的水声。

[雾·山雾] 雾从山谷里涌上来，你的脚先被吞掉，然后是膝盖，然后是腰。你站在原地不动，三秒后什么都看不见了，只有脚下的石头告诉你还在山上。空气里有松针和湿石头的味道，你深吸一口，凉意从鼻腔灌到肺底。

# 雪 · 3 entries

[雪·初雪] 一片雪花落在你的手背上，你看了一眼，是六角形的，还没看清就化了。第二片，第三片，越来越多，你的手背变成了一小片湿地。你抬头，雪花从灰白色的天幕里落下来，落在睫毛上，凉的，眨一下就没了。

[雪·大雪] 雪落在肩上，你的外套在十分钟内白了一层。你跺了跺脚，靴底踩进雪里，嘎吱一声，脚踝被雪埋了一半。风把雪粒吹进领口，凉得你缩脖子。你呼出的白气在面前散开，和雪花混在一起，分不清哪个是哪个。

[雪·融化] 屋檐上的雪在滴水，滴答滴答的节奏从头顶传下来。你踩在泥泞的路上，鞋底每走一步都带起一串泥点。空气里有湿土和残雪的凉意，你吸了一口，鼻腔发酸。阳光照在脸上是暖的，但风吹过来的时候，耳朵还是冻的。

# 冰雹 · 2 entries

[冰雹·打在身上] 第一颗冰雹砸在你的手背上，疼的，你缩了一下手。第二颗砸在肩膀上，比你想象的硬。你蹲下来抱住头，冰雹砸在后背上噗噗响，从衣领钻进去，凉得你打了个寒战。脚边的冰雹弹起来，打在小腿上，一颗一颗，密集得像有人在用指节敲你。

[冰雹·砸在屋顶] 冰雹砸在铁皮屋顶上，声音从噗噗变成叮叮当当，像有人在上面倒了一筐弹珠。你坐在屋里，声音大得盖住了电视。你走到窗边，冰雹在窗台上堆了一层，白色的，圆的，像缩小版的冰球。你伸手到窗外接了一颗，掌心被冰得发麻。

# 火山喷发 · 2 entries

[火山喷发·热浪] 热气从火山方向涌过来，你的脸被烤得发烫，像站在打开的烤箱前面。你往后退了两步，脚下的石头是热的，隔着鞋底也能感觉到。空气在抖，远处的树在热浪里扭曲成一片模糊的影子。你的眼睛干了，眨一下，眼角有沙粒磨的疼。

# 沙尘暴 · 2 entries

[沙尘暴·来袭] 风先到，沙后到。你眯起眼，沙粒打在脸上像有人在用砂纸蹭你的皮肤。嘴里进了沙子，牙齿咬下去嘎吱响。你把衣领拉起来捂住鼻子，呼吸透过布料，又闷又热。三米外的东西看不见了，你只能凭脚底的感觉判断方向。

[沙尘暴·过后] 你睁开眼，睫毛上挂着一层细沙，眨一下沙沙响。鼻腔里全是沙的味道，干的，苦的。你拍了拍头发，沙从发缝里落下来。空气还是浑的，你深吸一口，嗓子发紧，咳了两声。远处的太阳从灰黄色的天幕后面露出来，像一枚旧铜币。

# 龙卷风 · 1 entry

# 潮汐 · 2 entries

[潮汐·蓝眼泪] 夜里的浪打上来，海水发出蓝绿色的光。你蹲下来用手搅了一下水，光从指缝里涌出来，像液态的萤火虫。你的掌心被光照亮了，蓝色的，一闪一闪。你站起来，脚踩进水里，每一步都踩出一圈蓝光，像在踩碎一片星空。

# 云海 · 2 entries

[云海·山顶] 你站在山顶，脚下是白色的云，铺到天际线，看不见地面。你的大脑花了三秒才接受这个画面——你站在云的上面。风从背后推过来，你往前倾了一下，膝盖发软。云在你脚下流动，像一锅慢慢搅动的牛奶。

[云海·日出] 太阳从云海的边缘升起来，光把白色的云染成金橙色。你站在观景台上，冷风灌进衣领，后背起了一层鸡皮疙瘩。云在你脚下翻涌，太阳在你面前升起，你同时感到冷和暖。你的嘴唇干了，舔了一下，有铁锈的味道。

# 雾凇 · 1 entry

# 蓝眼泪 · 1 entry

# 火烧云 · 2 entries

[火烧云·傍晚] 云从西边烧起来，先是橘的，然后是红的，最后是紫的。你站在天台上，光打在脸上是暖的。你盯着看，云的边缘在变形，像什么东西在融化。你的脖子仰酸了，低下头，眼前一片绿——眼睛在适应，地面上的东西都带着一层绿色的残影。

[火烧云·倒映] 你站在江边，火烧云的倒影把水面染成铜红色。你蹲下来，手伸进水里，水是凉的，但颜色是暖的。远处有船经过，浪把倒影打碎了，红的、紫的、橘的碎片在水面上晃。你把手抽出来，水从指缝滴下去，每一滴都在江面上砸出一个小小的彩色漩涡。

[沙漠绿洲] 你走了六个小时，嘴唇裂了口子，嗓子干得咽不下口水。翻过最后一座沙丘，你看到棕榈树的绿色，膝盖软了一下。泉眼在沙丘脚下，你把脸浸进去，水凉得你打了个激灵。你的手指在水里泡着，皮肤上的沙被冲走了，指尖的褶皱比刚才又深了一层。蜻蜓从你头顶飞过去，翅膀在阳光里闪了一下。

[雨林树冠层] 你爬上四十米高的树冠平台，脚下是密密麻麻的绿叶，你站在绿色的海洋上面。风从树冠层吹过来，叶子在你脚下沙沙响。远处有鸟从一棵树飞到另一棵树，翅膀几乎碰到你的脸。阳光从叶缝里漏下来，光柱在绿叶上画出斑点，你的皮肤被晒得发烫，但风一吹又凉了。

[冻原永冻层] 你用铲子挖开地表的草皮，下面是冻了几千年的土，硬得像石头。你把铲子插进去，冰碴从缝隙里蹦出来打在你的脸上。土的颜色是灰蓝色的，你用手指摸了一下，凉意从指尖传到手肘。空气里有泥炭和冰混在一起的味道，你深吸一口，鼻腔发凉。地表的苔藓在你脚下软绵绵的，下面是永远化不开的冰。

[珊瑚白化] 你潜下去看，白的，像一片水下的坟场。珊瑚把共生的藻类吐出来，变成白色的骨架，你用手碰了一下，粗糙的碳酸钙刮手。鱼还在珊瑚之间游，但颜色淡了，像褪了色的照片。你的呼吸声在水下放大了，每一下都清清楚楚。你浮出水面，摘下面镜，眼睛酸了，阳光刺得你眨了两下。

[海蚀洞] 波浪几万年侵蚀出来的洞，你走进去，脚步声在石壁之间弹跳，回声比你的脚步慢了半拍。水从头顶滴下来，打在你的肩膀上，凉的。洞壁上有一层盐霜，你用手指刮了一下，白色的粉末落在你的鞋面上。洞的深处传来浪声，闷的，一下一下，像有人在石壁后面敲鼓。你往里走，光线暗了，空气潮得头发都在滴水。

[风蚀地貌] 你站在雅丹地貌的土丘之间，风从缝隙里灌过来发出呜呜声。土丘的形状像蘑菇，上面细下面粗，你用手摸了一下，沙粒粗粝得刮手。风把沙粒吹进眼睛，你眨了两下，眼角磨得疼。远处的土丘在热浪里扭曲成一片模糊的影子，你的影子比你高了三倍。空气里有干石头的矿物气味，你深吸一口，鼻腔干干的。

[冰锥] 屋檐上挂满了冰锥，你站在下面仰头看，最长的一根有半米。阳光照在冰锥上折射出彩虹的颜色，你眯起眼。冰锥的尖端在滴水，滴答滴答的节奏从头顶传下来。你伸手掰了一根下来，掌心被冰得发麻，冰锥在你手里慢慢融化，水从指缝滴下去落在雪地上，砸出一个个小坑。

[热泉] 你把手伸进热泉里，水烫得你缩回来，掌心红了一块。蒸汽从泉面升上来，你闻到一股硫磺味，呛得咳嗽。泉边的岩石上结着橙色的矿物沉淀，你用手指摸了一下，粗糙的颗粒刮手。远处的热泉在无风的空气里冒着白烟，你走近，热浪扑在脸上。脚下的地是温热的，你蹲下来把手插进去暖了一下，手掌湿了，冷风吹过又凉了。
