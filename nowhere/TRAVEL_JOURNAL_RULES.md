# 游记搜索与内容提升规则

## 目标
从真实游记中提取感官细节，补充和提升 Nowhere 内容库。

## 搜索策略

### 可用搜索渠道
- Bing 搜索（优先）
- 去哪儿/携程/马蜂窝游记页面
- 豆瓣游记/日记
- 小红书（如有开放页面）
- 知乎回答

### 搜索关键词模板
```
{城市名} 游记 街道 声音 气味
{城市名} 旅行 细节 感受
{城市名} 漫步 小巷 发现
{城市名} 当地人 生活 日常
{city} travel journal sensory details
{city} street sounds smells local life
```

## 提取规则

### 必须提取的5类细节
1. **声音**：具体是什么声音、从哪里来、什么节奏
   - ❌ "有音乐声"
   - ✅ "清真寺的宣礼声从扩音器里出来，在巷子里回响"

2. **气味/味道**：具体是什么味道、混合了什么、从哪里飘来
   - ❌ "有食物的香味"
   - ✅ "烤羊肉的烟从铁签子上冒出来，孜然的味道混着炭火的焦"

3. **触感**：手/脚/皮肤碰到什么、温度、质地
   - ❌ "石头很凉"
   - ✅ "你把手贴在石墙上，凉的，指尖碰到苔藓的湿"

4. **具体地标**：只有那个地方才有的东西
   - ❌ "一家小店"
   - ✅ "李子坝轻轨站——轻轨从楼体中间穿过去"

5. **当地人的行为**：具体动作、习惯、只有那里才有的事
   - ❌ "有人在喝茶"
   - ✅ "老人把盖碗茶的茶盖掀开一条缝，吹了吹，又盖上"

### 提取格式
```json
{
  "city": "城市名",
  "details": [
    {
      "type": "声音|气味|触感|地标|行为",
      "text": "具体描述（用第二人称'你'）",
      "source": "游记来源（可选）"
    }
  ]
}
```

## 写入规则

### 质量标准（6条铁律）
1. 有"不该在这里的东西"——百科全书没有的细节
2. 声音有层次——给声音一个比喻或情绪
3. 时间在流动——"然后..."/"比刚才..."
4. 你和它互相看了——两个存在之间的关系
5. 你做了什么的反作用——小动作让"你"在场
6. 停在描述上就结束 = 弱——多走一步

### 禁词（零容忍）
很 / 非常 / 十分 / 重大 / 巨大 / 美丽 / 壮观 / 震撼 / 壮丽 / 宏伟

### 风格偏好
- 描述性写法 > 判断式短句
- "人的语气" > "代码的语气"
- 第二人称现在时："你蹲下来"，不是"旅行者蹲下来"
- 短句：10-20字
- 身体感：皮肤/脚底/舌头/鼻腔/耳朵

## 工作流程

### 每个城市的处理步骤
1. 用搜索引擎搜索该城市游记（至少3篇）
2. 提取5类细节（声音/气味/触感/地标/行为）
3. 检查现有内容是否已有类似细节
4. 用提取的细节补充或重写内容
5. 质量检查（6条铁律+禁词）
6. 提交

### 优先级
1. encounters 文件（最需要游记支撑）
2. scene_soundscape/taste（已有游记来源，需要补充更多城市）
3. localcolor（需要具体细节）
4. seasonal（需要季节性细节）

## 质量参照（北京/重庆范例）

### 声景范例（来自 scene_soundscape.txt）

**北京：**
```
[北京] 鸽哨从胡同上方掠过，嗡嗡的哨音在灰墙之间回旋。你抬头，看不见鸽子，只有声音在瓦片上方打转，一圈一圈淡下去。
```
→ 好在哪里：具体声音（鸽哨）、具体地点（胡同灰墙）、声音有形状（打转、淡下去）

**重庆：**
```
[重庆] 轻轨从李子坝的楼体中间穿过，轰隆声从头顶压下来。你站在观景平台上，列车的风掀动你的衣角，铁轨的震动从脚底一直传到牙齿。
```
→ 好在哪里：只有重庆才有的地标（李子坝轻轨穿楼）、触感具体到牙齿

### 味觉范例（来自 scene_taste.txt）

**北京：**
```
[北京] 冬天的胡同里，煤球炉子烧着，烟从烟囱口冒出来，呛的，带一点硫磺味。你缩着脖子走过去，烤红薯的铁皮桶炉子支在墙根，白气从桶口冲出来，焦甜的。你买了一个，掰开，橘色的瓤烫手，糖汁流到指缝里。你吹了两口咬下去，甜到舌根，纤维塞在牙缝里。
```
→ 好在哪里：混合气味（硫磺+焦甜）、身体动作具体到衣服（缩着脖子）、意外细节（纤维塞在牙缝里）

**重庆：**
```
[重庆] 防空洞改成的火锅店，你走进去，洞壁是粗糙的岩壁，被油烟熏得发黑发亮。牛油在锅里翻滚，辣味在封闭的空间里浓得化不开。你涮了一片鸭肠，脆的，嚼的时候能听到咯吱声。洞里有回音，旁边桌划拳的声音在头顶滚来滚去。你出了一身汗，衣服贴在后背上。
```
→ 好在哪里：只有重庆才有的地方（防空洞火锅店）、声音有层次（嚼鸭肠的咯吱+划拳的回音）、身体感受（衣服贴在后背上）

### 从范例提炼的规律
1. **具体地标**：李子坝、防空洞火锅店、鹤鸣茶社——只有那个地方才有的
2. **混合感官**：不是单一的"有味道"，是硫磺+焦甜、辣味+划拳声
3. **身体到衣服**：不只是"你感到"，是"衣服贴在后背上"、"糖汁流到指缝里"
4. **意外收尾**：纤维塞在牙缝里、分不清是习惯了还是真觉得好喝

## 注意事项
- 不要直接复制游记原文，要用第二人称改写
- 细节必须真实可信，不要编造
- 每个城市至少提取5个有效细节
- 提取后检查是否与现有内容重复
